## 什么是 Neo-Hexo？

Neo-Hexo 是一个现代化的静态站点生成器，基于经典的 Hexo 框架重新设计，采用全新的架构：

- **纯 ESM** — 告别 CommonJS，拥抱现代 JavaScript
- **TypeScript 原生** — 完整的类型安全
- **插件化架构** — 基于 Hook 系统的可扩展设计
- **Edge.js 模板** — 现代化的模板引擎，支持组件和布局继承

## 快速开始

创建一篇新文章：

```bash
neo-hexo new post "My First Post"
```

启动开发服务器：

```bash
neo-hexo server
```

生成静态文件：

```bash
neo-hexo generate
```
