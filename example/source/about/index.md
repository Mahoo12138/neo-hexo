## 关于这个站点

这是一个 Neo-Hexo 示例站点，用于演示框架的核心功能。

## 关于 Neo-Hexo

Neo-Hexo 是经典 [Hexo](https://hexo.io) 静态博客框架的现代化重构版本。

### 核心改进

- **纯 ESM + TypeScript** — 从零开始重写，完整类型安全
- **插件化架构** — 基于 Hook 系统，所有功能皆为插件
- **现代模板引擎** — Edge.js 替代 Nunjucks/EJS
- **内置开发服务器** — WebSocket 热重载 + 错误覆盖层
- **声明式配置** — `neo-hexo.yaml` 一站式配置

### 技术栈

| 组件 | 技术选型 |
|------|---------|
| 运行时 | Node.js ≥ 22 |
| 语言 | TypeScript (strict) |
| 模块 | 纯 ESM |
| 模板 | Edge.js |
| Markdown | markdown-it + shiki |
| 包管理 | pnpm workspace |
| 测试 | Vitest |
